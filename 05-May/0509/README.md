# 위장

문제 링크 : [위장](https://programmers.co.kr/learn/courses/30/lessons/42578)

### 1.이해

- 매일 다른 옷을 조합하여 위장한다.
- 의상들이 담긴 배열 `clothes`에서 서로 다른 옷의 조합의 수를 return 한다.
- clothes의 각 행은 `[의상의 이름, 의상의 종류]` 로 이루어져 있다.
- 같은 이름의 의상은 없다.
- 하루 최소 한 개의 의상은 입는다.
