# 정수 내림차순으로 배치하기

[정수 내림차순으로 배치하기](https://programmers.co.kr/learn/courses/30/lessons/12933)

### 1. 이해

- 정수 n을 내림차순으로 배치한다.

### 2. 계획

- n을 String으로 바꿈
- `match[/[0-9/g]]`로 각 자리수를 원소로 갖는 배열 뽑아냄.
- 각 원소는 string 이므로 `map` 메서드를 이용해 각 원소를 `Number()`사용해서 바꿈.
- 바뀐 배열을 `sort()` 메서드를 이용해서 내림차순 정렬함.
